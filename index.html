JSON to Table Comparison Tool

Instructions:

  1. Upload or paste JSON/CSV data into the text areas for both tables.
  2. Click "Update Table 1" and "Update Table 2" to populate the tables.
  3. Select the columns to compare using the checkboxes.
  4. Click "Compare" to highlight the differences between the tables.
  5. Use "Clear Highlight" to remove the highlights.
  6. Toggle parallel scrolling on and off using the "Toggle Parallel Scrolling" button.
